# 🏏 IPL Analysis Dashboard

An interactive data analytics dashboard built using **Python** and **Streamlit** to explore, analyze, and visualize Indian Premier League (IPL) data. The project provides valuable insights into team performances, player statistics, match outcomes, venue trends, and season-wise analysis through dynamic visualizations.

## 📌 Project Overview

The IPL Analysis Dashboard helps cricket enthusiasts, analysts, and data science learners gain insights from historical IPL datasets using Exploratory Data Analysis (EDA) and interactive visualizations.

## 🚀 Features

* Team Performance Analysis
* Season-wise IPL Statistics
* Top Run Scorers Analysis
* Top Wicket Takers Analysis
* Toss Impact Analysis
* Venue-wise Performance Insights
* Winning Team Trends
* Interactive Filters and Visualizations
* User-Friendly Streamlit Interface
